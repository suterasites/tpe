# Total Piling & Excavations - Mockup Assets

Drop the assets you want used in the homepage mockup into the relevant subfolder, then ping me to start the build.

## Folder structure

| Folder | What goes in here | Notes |
|---|---|---|
| `logo/` | Logo (SVG ideal, PNG with transparency fine) | One light-bg version + one dark-bg version if you have both. Used in nav + footer. |
| `hero/` | Hero background photo | Wide landscape, ideally 2400x1200+ at 80% quality JPG. Excavator on site / screw pile install / ground works vibe. |
| `services/` | One photo per service - 4 total | Square or 4:3, ~1200x1200. Name them `screw-piling.jpg`, `underpinning.jpg`, `shoring.jpg`, `sales.jpg` so I can wire them straight into the cards. |
| `trust/` | Australian Made badge + A.A. Steel Works logo | PNG with transparency. These come straight off their current site if you don't have originals - I can pull them from the Wix CDN. |
| `projects/` | Optional - any completed-job photos | Used in a "Recent Work" gallery if you want one. Skip if not. |

## Constraints

- No watermarks, no source-credit overlays.
- No fake social proof (no fake testimonial avatars, no fake review screenshots).
- If a photo is stock / not their actual work, flag it in the filename (e.g. `screw-piling-stock.jpg`) so we don't accidentally claim it as their job.

## When ready

Tell me the assets are in and I'll start the build. If you're missing a slot (e.g. no service photo for one tile), I'll fall back to a clean text-only card for that one rather than padding with stock.
